# ihcux-prot-baixa-fidelidade-tela-login-v1

## Projeto: Tela de Login (Miro)

## Descrição

Protótipo simples de uma tela de login desenvolvido no Miro, com foco em estrutura e organização de interface.

## Objetivo

Praticar conceitos básicos de UI e prototipagem.

## Elementos da Tela

* Ícone de usuário
* Campo de login
* Campo de senha
* Botão de entrada

## Ferramenta Utilizada

* Miro

## Estrutura

Tela de Login composta por campos de entrada e botão de acesso.
